# 示例文件
`vim application.properties`
```bash
# 文件内容
server.port=8080
server.name=app
spring.redis.host=localhost
```

yy 复制当前行
p 粘贴
u 撤销
ctrl+r 恢复
ciw (change in the word)
diw (delete in the word)
set number 显示行数
n(数字) 去第n行
G 跳到文件结尾
gg 跳到文件开头
