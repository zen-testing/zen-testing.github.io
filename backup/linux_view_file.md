# head

-n 看前n行

# tail

-n 看后n行
-f 循环打印

# less

方向键换行
空格下一页
ctrl + b 恢复
-N 显示行号
+/word 高亮显示关键词

# more

默认展示第一屏
回车下一行
空格翻页
-n x 每次回车  显示x行 ctrl + b 恢复

# cat

默认打印所有内容
-n 带行号打印
