在[网站](https://store.rg-adguard.net/)通过`PackageFamilyName`查找`Microsoft.WindowsStore_8wekyb3d8bbwe`
下载所有符合条件的appx
按照主程序在最后的顺序依次安装
提示被占用就可以跳过
